# 方案标题
鱼食自动投喂器
# 应用场景
适用于家庭自动鱼食投喂，解决人为投喂烦恼
# 功能介绍
1. 自动检测鱼食余量，低于阈值可主动通知手机app端
2. 定期进行鱼食自动投放，可解决家中无人投食情况
3. 可手机app端远程投食一定量的鱼食，远程查看鱼缸情况
4. 可加入自动换水装置，远程启动排水换水
5. 可增加弹射鱼食模式，在鱼缸范围内随机弹射鱼食
6. 设备低电量时，启动告警通知
